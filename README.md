# 質悅 ZHI YUE Brand Concept

This is a one-page Next.js concept site generated from `docs/research/zhiyue-brand-research-report.pdf`.

## Purpose

- Create a mobile-first brand entrance for ZHI YUE / 質悅.
- Present the Gallery Calm direction from the research report.
- Keep all unverified business information in data files instead of hard-coding it across components.

## Local Setup

```bash
pnpm install
pnpm dev
pnpm build
```

## Content And Rights

- Images in `public/images/generated` are AI concept visuals.
- They are not real salon photos, customer work, stylist portraits, or official brand assets.
- Formal launch requires brand confirmation of copy, opening hours, booking links, logo usage, and image authorization.

## Known Pending Items

- Official LINE or online booking URL.
- Single confirmed opening-hours version.
- Approved logo and brand usage rules.
- Approved stylist list, service pricing, and portfolio images.
- Official app download URL, if an app exists.
